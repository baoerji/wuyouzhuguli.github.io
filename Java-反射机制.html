Java 反射机制

Posted on 2016-09-16

JAVA反射机制是在运行状态中，对于任意一个类，都能够知道这个类的所有属性和方法；对于任意一个对象，都能够调用它的任意一个方法和属性；这种动态获取的信息以及动态调用对象的方法的功能称为java语言的反射机制。—-百度百科

动态：运行期间动态绑定执行规则。

静态：编译以后就已经确定的执行过程。
